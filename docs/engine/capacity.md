# 产能计算

## 初始产能公式

```text
日可产数量 = UPPH × 工作小时
```

示例：

```text
UPPH = 100
工作小时 = 10
日可产数量 = 1000
```

如果订单数量为 2500，则排产：

```text
Day 1: 1000
Day 2: 1000
Day 3: 500
```

## 后续扩展公式

如果 UPPH 是人均 UPPH，则扩展为：

```text
日可产数量 = 人均 UPPH × 工作小时 × 人数 × 效率系数 × 良率系数
```

## UPPH 缺失处理

对于缺少 UPPH 的工段：

1. 保留工段节点。
2. 标记为产能异常。
3. 不自动跳过。
4. 可由人工确认默认产能或放行。
