# 排产引擎总览

第一版排产引擎建议采用：

```text
规则 + 贪心
```

目标是：

- 结果稳定
- 逻辑可解释
- 便于调试
- 易于扩展

## 基础流程

```text
1. 创建排产批次 runId
2. 读取候选订单
3. 执行订单筛选规则
4. 按优先级排序，初期按 DlvDate ASC
5. 对每个订单查询工艺路线
6. 按 OperationSeq 拆解工段
7. 对每个工段匹配候选产线
8. 计算候选产线最早可完成日期
9. 选择最早可完成或优先级最高产线
10. 按日期逐日占用产能
11. 生成 schedule_results
12. 生成 schedule_exceptions 与 schedule_rule_trace
13. 汇总订单开始日、完成日、是否逾期、瓶颈工段
```
