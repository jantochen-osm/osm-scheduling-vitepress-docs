# 系统目标范围

## 初始范围

| 项目 | 初始选择 |
| --- | --- |
| 排产对象 | MO 制造订单 |
| 排产范围 | OSM 产线 |
| 排产粒度 | MO + 工段 + 日期 + 产线 |
| 排产方式 | 按 DlvDate 正序的日级正排 |
| 数据来源 | 聚合 Dynamics AX 2012 后的标准化数据 |
| 规则承载 | 规则中心，后续可基于 NocoBase 维护规则输入表 |

## 初始不覆盖范围

第一版不建议覆盖：

- 小时级精细排程
- 复杂数学优化求解器
- 物料齐套自动约束
- 多线并行拆单
- 分批流转
- 自动换线损耗优化
- AI 自动生成最终计划且无人工审核

## 初始核心假设

```text
同一 MO 的同一工段允许跨日，不允许跨线。
前工段整单完成后，后工段才允许开始。
缺少关键数据时进入异常清单，不自动猜测。
```
