# 排产结果评价指标

| 指标 | 说明 | 阶段 |
| --- | --- | --- |
| 订单预计开始日 | 订单第一个工段开始日期 | 阶段 1 |
| 订单预计完成日 | 订单最后一个工段完成日期 | 阶段 1 |
| 是否逾期 | FinishDate > DlvDate | 阶段 1 |
| 逾期天数 | FinishDate 与 DlvDate 的日期差 | 阶段 1 |
| 产线负荷率 | 已排工时 / 可用工时 | 阶段 4 |
| 瓶颈工段 | 导致订单完成延迟或产能不足的关键工段 | 阶段 4 |
| 异常订单数 | 按异常类型统计 | 阶段 1~4 |
| 规则命中率 | 有明确规则匹配的订单比例 | 阶段 4 |
| 人工调整次数 | 计划员对自动结果的修改次数 | 阶段 5 |
| 版本差异 | 不同计划版本之间完成日、负荷、逾期的差异 | 阶段 5 |
