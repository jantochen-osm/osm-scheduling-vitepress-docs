# 规则优先级

建议统一规则优先级如下：

```text
人工锁定/人工指定
> 试产/量产限制
> FG Item Code 专线规则
> Customer / Brand 专线规则
> 工段类型规则
> OSM_Category 分类规则
> 默认产线规则
```

## 冲突处理原则

当多个规则同时命中时：

1. 判断是否互斥。
2. 如果互斥且无法通过优先级解决，进入规则冲突异常。
3. 如果可通过优先级解决，记录最终采用规则与被覆盖规则。

## 示例

```text
IF FG Item Code 命中 3F2 专线规则
AND 工段类型 = Assembly
THEN 优先排 3F2
ELSE 再考虑 3F3~3F6
```
