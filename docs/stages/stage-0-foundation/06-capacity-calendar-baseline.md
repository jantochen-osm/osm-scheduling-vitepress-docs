# 06 产能与日历口径

产能与日历口径用于回答：**某条线某一天到底能生产多少。**

## 1. 初始产能公式

Stage 1 建议使用最小公式：

```text
日可产数量 = UPPH × 工作小时
```

例如：

```text
UPPH = 100
工作小时 = 10
日可产数量 = 1000
```

## 2. UPPH 口径确认

Stage 0 必须确认 UPPH 是哪一种：

| UPPH 类型 | 含义 | 对公式的影响 |
| --- | --- | --- |
| 线体级 UPPH | 整条线每小时产出 | 日产能 = UPPH × 工时 |
| 人均 UPPH | 每人每小时产出 | 日产能 = UPPH × 工时 × 人数 |
| 产品级 UPPH | 某产品标准小时产出 | 需要结合产线或工段 |
| 工段级 UPPH | 某工段标准小时产出 | 需要结合产品和产线 |

::: warning 阶段 0 必须明确
如果 UPPH 口径不明确，Stage 1 的产能结果可能放大或缩小数倍。
:::

## 3. 工作日历规则

Stage 0 建议先确认以下规则：

| 日期类型 | 默认工时 | 说明 |
| --- | --- | --- |
| 正常工作日 | 10h | 默认标准工作日 |
| 周六 | 8h | 默认短工时 |
| 周日 | 0h | 初期默认不排产 |
| 大型节假日前一天 | 8h | 默认短工时 |
| 每月最后一个工作日 | 8h | 默认短工时 |
| 法定节假日 | 0h | 默认不排产 |
| 产线级特殊日历 | 按维护值 | 优先级高于通用规则 |

## 4. 日历优先级

建议优先级：

```text
产线级手工停线 / 加班
> 法定节假日
> 大型节假日前一天
> 每月最后一个工作日
> 周六
> 正常工作日
```

## 5. Stage 1 最低日历要求

Stage 1 只需要：

```text
正常工作日 = 10h
```

但 Stage 0 应先明确周六、周日、节假日，避免 Stage 2/4 返工。

## 6. 产能异常

| 异常类型 | 触发条件 | 处理方式 |
| --- | --- | --- |
| MISSING_UPPH | 找不到 UPPH | 阻断或进入人工确认 |
| ZERO_WORK_HOURS | 当天工时为 0 | 跳过该日期 |
| MISSING_CALENDAR | 找不到日历 | 使用默认规则或进入异常，需业务确认 |
| INVALID_UPPH | UPPH 小于等于 0 | 阻断产能计算 |
| LINE_NOT_SCHEDULABLE | 产线不可排 | 不作为候选线 |

## 7. 输出物

建议输出：

```text
stage0_capacity_definition.md
stage0_work_calendar.xlsx
stage0_calendar_exception_rules.xlsx
```
