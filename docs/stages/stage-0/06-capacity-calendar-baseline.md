# 06 产能与日历口径

产能与日历口径用于回答：**某条线某一天到底能生产多少。**

## 1. 初始产能公式

UPPH 已确认为**人均每小时产出**，因此产能公式需要引入人数：

```text
日可产数量 = UPPH × 人数 × 工作小时
```

例如：

```text
UPPH = 100（人均每小时产出）
人数 = 5
工作小时 = 10
日可产数量 = 100 × 5 × 10 = 5000
```

## 2. UPPH 口径（已确认）

::: tip 已确认
UPPH = **人均每小时产出**（Units Per Person Per Hour）
:::

产能公式：

```text
日可产数量 = UPPH × 人数 × 工作小时
```

这意味着系统需要额外维护**每条产线每个班次的排班人数**。

| 需要的数据 | 来源 | 说明 |
| --- | --- | --- |
| UPPH | 产能主数据 | 人均每小时产出，粒度：产线 × 产品 × 工段 |
| Headcount（人数） | 排班主数据 | 产线当天的实际排班人数 |
| WorkHours（工作小时） | 工作日历 | 当天可用生产工时 |

## 3. 工作日历规则

Stage 0 建议先确认以下规则：

| 日期类型 | 默认工时 | 说明 |
| --- | --- | --- |
| 正常工作日 | 10h | 默认标准工作日 |
| 周六 | 8h | 默认短工时 |
| 周日 | 0h | 初期默认不排产 |
| 大型节假日前一天 | 8h | 默认短工时 |
| 每月最后一个工作日 | 8h | 默认短工时 |
| 法定节假日 | 0h | 默认不排产 |
| 产线级特殊日历 | 按维护值 | 优先级高于通用规则 |

## 4. 日历优先级

建议优先级：

```text
产线级手工停线 / 加班
> 法定节假日
> 大型节假日前一天
> 每月最后一个工作日
> 周六
> 正常工作日
```

## 5. Stage 1 最低日历要求

Stage 1 只需要：

```text
正常工作日 = 10h
```

但 Stage 0 应先明确周六、周日、节假日，避免 Stage 2/4 返工。

## 6. 产能异常

| 异常类型 | 触发条件 | 处理方式 |
| --- | --- | --- |
| MISSING_UPPH | 找不到 UPPH | 阻断或进入人工确认 |
| ZERO_WORK_HOURS | 当天工时为 0 | 跳过该日期 |
| MISSING_CALENDAR | 找不到日历 | 使用默认规则或进入异常，需业务确认 |
| INVALID_UPPH | UPPH 小于等于 0 | 阻断产能计算 |
| LINE_NOT_SCHEDULABLE | 产线不可排 | 不作为候选线 |
