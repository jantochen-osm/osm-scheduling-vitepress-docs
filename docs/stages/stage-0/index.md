# 阶段 0：业务边界与数据底座确认

## 阶段简介

阶段 0 是整个 OSM 排产系统建设的**起点**，不产出任何排产代码，只聚焦于：

- 确认系统**排什么**、**用什么数据排**、**排到什么粒度**
- 准备可供阶段 1 直接使用的**数据样本**与**字段清单**

::: tip 核心原则
阶段 0 的产出是文档与数据样本，不是代码。**阶段 0 未完成时，阶段 1 不应启动。**
:::

## 快速概览

| 项目 | 内容 |
| --- | --- |
| 阶段目标 | 确认业务边界、梳理数据字段、准备样例数据集 |
| 产出物 | 字段清单、规则清单、异常清单、样例数据集、MVP 范围确认 |
| 对应版本 | V0.1 数据样板版 |
| 主要风险 | AX 字段口径不一致、UPPH 口径不清 |
| 阶段门禁 | 样例数据可手工验证至少 1 条完整排产流程 |
